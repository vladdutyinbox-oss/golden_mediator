**Микросервис: `rust-inference`**

**Роль в событийной цепочке `user-message-processing`**  
Независимый узел **тяжёлого инференса**.  
- **Вход**: потребляет событие `LLMInferenceRequested` из Kafka.  
- **Действие**: Rust-ядро выполняет инференс (Ollama + TensorRT-LLM fallback).  
- **Выход**: публикует поток `LLMResponseChunkGenerated` (streaming) + финальное `LLMResponseGenerated`.  
- **Полная независимость**: сервис запускается/останавливается/масштабируется **отдельно** от rag-service, inference-router и response-processor. Все внешние зависимости (Kafka, Ollama, Model-Selection) заменяются моками.

### 1. Устройство репозитория (DDD + Vertical Clean Slices)

Репозиторий построен по **Domain-Driven Design** + **Vertical Slice Architecture** (каждый use-case — отдельный вертикальный слайс).  
Чистая архитектура (Onion): Domain → Application → Infrastructure → Adapters.

### 3. Тесты с моковыми заглушками (все интеграции)

Все тесты **изолированные** — запускаются без Kafka, без Ollama, без других микросервисов.

**Моковые заглушки** (в `tests/mocks/`):

| Интеграция              | Мок-библиотека (Rust)          | Что мокается                              | Измеряемый показатель в тесте                  |
|-------------------------|--------------------------------|-------------------------------------------|------------------------------------------------|
| Kafka consumer          | `testcontainers` + `rdkafka` mock | `LLMInferenceRequested`                   | Время от consume до первого chunk (p95 < 50 мс) |
| Kafka producer          | `mockito` / `wiremock`         | `LLMResponseChunkGenerated` + final event | Кол-во опубликованных событий (streaming)     |
| Ollama / TensorRT-LLM   | `httpmock`                     | `/api/generate` streaming ответ           | Latency inference (p95 < 320 мс)<br>Tokens/s (> 60) |
| OpenTelemetry           | `opentelemetry::sdk::testing`  | spans + metrics                           | Кол-во span'ов + histogram latency            |
| Model selection         | in-memory Redis mock           | `mock:conversation:{id}:model`            | Правильный выбор модели                       |

**Пример интеграционного теста** (`tests/integration/inference_flow_test.rs`):

```rust
#[tokio::test]
async fn test_full_inference_flow_with_mocks() {
    // 1. Мокаем Kafka
    let kafka_mock = KafkaMock::new().expect_message("LLMInferenceRequested");
    
    // 2. Мокаем Ollama
    let ollama_mock = httpmock::MockServer::start();
    ollama_mock.mock(|when, then| {
        when.method(POST).path("/api/generate");
        then.status(200).body_streaming(chunk_stream()); // 10 чанков
    });

    // 3. Запускаем сервис в тестовом режиме
    let service = InferenceService::with_mocks(
        kafka_mock.consumer(),
        ollama_mock.url(),
        MetricsRegistry::new(),
    );

    let start = Instant::now();
    service.handle_command(mock_command()).await;

    // === ИЗМЕРЯЕМЫЕ ПОКАЗАТЕЛИ ===
    let duration = start.elapsed();
    assert!(duration.as_millis() < 320, "p95 latency violation"); // из резюме

    let metrics = service.metrics();
    assert!(metrics.tokens_per_second() > 60.0);
    assert!(metrics.vram_usage_gb() < 8.0);
    assert_eq!(metrics.chunks_published(), 10);
    assert!(metrics.jaeger_span_count("inference") == 3); // preprocess → infer → post
}
```

#### `src/main.rs`
```rust
use actix_web::{web, App, HttpResponse, HttpServer, Responder};
use std::env;
use tokio::sync::mpsc;

mod domain;
mod application;
mod infrastructure;

#[actix_web::main]
async fn main() -> std::io::Result<()> {
    tracing_subscriber::fmt().init();

    let kafka_servers = env::var("KAFKA_BOOTSTRAP_SERVERS").unwrap_or("localhost:9092".to_string());
    let ollama_host = env::var("OLLAMA_HOST").unwrap_or("http://localhost:11434".to_string());
    let default_model = env::var("MODEL_NAME").unwrap_or("phi3:mini".to_string());

    // Запуск потребителя Kafka в фоне
    let (tx, mut rx) = mpsc::channel(100);
    tokio::spawn(async move {
        infrastructure::kafka::consumer::start_consumer(&kafka_servers, tx).await;
    });

    // Обработка событий
    tokio::spawn(async move {
        while let Some(cmd) = rx.recv().await {
            let result = application::inference_slice::service::handle_inference(
                cmd,
                &ollama_host,
                &default_model,
            ).await;
            if let Ok(events) = result {
                infrastructure::kafka::producer::publish_events(&kafka_servers, events).await;
            }
        }
    });

    // Простой HTTP для теста (без Kafka)
    HttpServer::new(move || {
        App::new()
            .app_data(web::Data::new(ollama_host.clone()))
            .app_data(web::Data::new(default_model.clone()))
            .route("/health", web::get().to(health))
            .route("/infer", web::post().to(infer_test))
    })
    .bind("0.0.0.0:8081")?
    .run()
    .await
}

async fn health() -> impl Responder {
    HttpResponse::Ok().body("rust-inference OK (CPU mode)")
}

async fn infer_test(
    req: web::Json<domain::inference::model::InferenceRequest>,
    ollama_host: web::Data<String>,
    model: web::Data<String>,
) -> impl Responder {
    let cmd = domain::inference::model::LLMInferenceCommand {
        request_id: uuid::Uuid::new_v4().to_string(),
        conversation_id: "test-conv".to_string(),
        model: model.to_string(),
        prompt: req.prompt.clone(),
    };

    match application::inference_slice::service::handle_inference(
        cmd,
        &ollama_host,
        &model,
    ).await {
        Ok(events) => HttpResponse::Ok().json(events),
        Err(e) => HttpResponse::InternalServerError().body(e.to_string()),
    }
}
```

#### `src/application/inference_slice/service.rs`
```rust
use crate::domain::inference::{model::LLMInferenceCommand, event::InferenceEvent};
use crate::infrastructure::ollama_client;

pub async fn handle_inference(
    cmd: LLMInferenceCommand,
    ollama_host: &str,
    model: &str,
) -> Result<Vec<InferenceEvent>, String> {
    let mut events = Vec::new();
    let mut stream = ollama_client::generate_stream(ollama_host, model, &cmd.prompt).await?;

    while let Some(chunk) = stream.recv().await {
        events.push(InferenceEvent::LLMResponseChunkGenerated(chunk.clone()));
        if chunk.done {
            events.push(InferenceEvent::LLMResponseGenerated(chunk));
            break;
        }
    }
    Ok(events)
}
```

#### `src/infrastructure/ollama_client.rs`
```rust
use crate::domain::inference::model::LLMResponseChunkEvent;
use tokio::sync::mpsc;
use reqwest::Client;
use futures_util::StreamExt;

pub async fn generate_stream(
    host: &str,
    model: &str,
    prompt: &str,
) -> Result<mpsc::Receiver<LLMResponseChunkEvent>, String> {
    let client = Client::new();
    let payload = serde_json::json!({
        "model": model,
        "prompt": prompt,
        "stream": true
    });

    let res = client.post(format!("{}/api/generate", host))
        .json(&payload)
        .send()
        .await
        .map_err(|e| e.to_string())?;

    let (tx, rx) = mpsc::channel(50);
    tokio::spawn(async move {
        let mut stream = res.bytes_stream();
        while let Some(chunk) = stream.next().await {
            if let Ok(bytes) = chunk {
                if let Ok(text) = String::from_utf8(bytes.to_vec()) {
                    if let Ok(json) = serde_json::from_str::<serde_json::Value>(&text) {
                        let chunk_event = LLMResponseChunkEvent {
                            request_id: "stream-1".to_string(),
                            chunk: json["response"].as_str().unwrap_or("").to_string(),
                            done: json["done"].as_bool().unwrap_or(false),
                        };
                        let _ = tx.send(chunk_event).await;
                    }
                }
            }
        }
    });

    Ok(rx)
}
```

#### `src/infrastructure/kafka/consumer.rs`
```rust
use rdkafka::consumer::{StreamConsumer, Consumer};
use rdkafka::ClientConfig;
use tokio::sync::mpsc;
use crate::domain::inference::model::LLMInferenceCommand;

pub async fn start_consumer(brokers: &str, tx: mpsc::Sender<LLMInferenceCommand>) {
    let consumer: StreamConsumer = ClientConfig::new()
        .set("group.id", "rust-inference-group")
        .set("bootstrap.servers", brokers)
        .set("auto.offset.reset", "earliest")
        .create()
        .expect("Failed to create consumer");

    consumer.subscribe(&["inference.requests"]).unwrap();

    while let Ok(msg) = consumer.recv().await {
        if let Some(payload) = msg.payload() {
            if let Ok(cmd) = serde_json::from_slice::<LLMInferenceCommand>(payload) {
                let _ = tx.send(cmd).await;
            }
        }
    }
}
```

#### `src/infrastructure/kafka/producer.rs`
```rust
use rdkafka::producer::{FutureProducer, FutureRecord};
use rdkafka::ClientConfig;
use crate::domain::inference::event::InferenceEvent;

pub async fn publish_events(brokers: &str, events: Vec<InferenceEvent>) {
    let producer: FutureProducer = ClientConfig::new()
        .set("bootstrap.servers", brokers)
        .create()
        .expect("Failed to create producer");

    for event in events {
        let payload = serde_json::to_string(&event).unwrap();
        let _ = producer.send(
            FutureRecord::to("inference.responses")
                .payload(&payload)
                .key("inference"),
            std::time::Duration::from_secs(0),
        ).await;
    }
}
```
