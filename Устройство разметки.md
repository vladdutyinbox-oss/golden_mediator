

## Разметка

### Введение

Приложение представляем узлом в сети взимодействий. \
Разбиваем устройство на 3 уровня, которые также \
являются самодостаточными подобными узлами.\
Каждый уровень делится на разделы: Описание, показатели\
, Действия.

Уровни: 
- целое приложение в сети пользователей и \ 
разработчиков с описанием лучших взаимодействий
- Описание цепочек взаимодействий с разбором \
прохода по узлам приложения
- Описание узлов приложения (микросервисов)

Разделы: 
- Описание. Покзаывает особенности и свойства \
взаимодействий с последовательным углублением \
в тонкости устройства с проходом вниз по уровням
- Показатели. Это меры, по которым можно оценить\
свойства взаимодействий как стомость, отталкиваясь\
от текущего уровня глубины рассмотрения.
- Действия. Описание того, какие взаимодекйствия \
мы можем провести с текущим уровнем как с \
самодостаточным узлом. Это действия исследования,\
создания и поддержки.

### Правила

Целое приложение
- как узел в сети

Цепочки взаимодействий
- - как события


Узлы устройства

- по влиянию на 3 уровня устройства - Узел, Цепочка, \
Поведение прилодения
